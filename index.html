Skills & Projects

Creator of Wellness and Breathing App: Breathe Sync

I created Breathe Sync, a breathing timer application designed to help users improve their well-being through guided breathing exercises. The app was developed using Swift programming language and Xcode development environment, allowing me to create an intuitive and responsive user interface.

After rigorous testing and refinement, I successfully launched Breathe Sync on the Apple App Store, making it accessible to users worldwide. This project showcases my ability to take an idea from conception to completion, utilizing my programming skills to create a practical solution for wellness and mindfulness.
